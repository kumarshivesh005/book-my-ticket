# 🎟️ Book My Ticket (Backend)

A simplified movie ticket booking backend system built by extending an existing codebase.  
This project implements authentication, protected routes, and safe seat booking using PostgreSQL.

---

## 🚀 Features

- 👤 User Registration
- 🔐 User Login with JWT Authentication
- 🛡️ Protected Booking Routes
- 🎫 Seat Booking System
- ❌ Prevent Duplicate Seat Booking
- 🔄 Transaction-based Booking (PostgreSQL)
- 🧱 Clean Folder Structure (MVC Pattern)

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- PostgreSQL
- bcryptjs (Password Hashing)
- jsonwebtoken (JWT Auth)
